## Installation Methods

<Tabs>
  <Tab title="Interactive Menu">
    The primary installation method uses an interactive menu system:

    ```bash
    bash -c "$(wget -qLO - https://github.com/tteck/Proxmox/raw/main/ct/scriptname.sh)"
    ```

    This presents a dialog-based interface for configuring:
    - Container resources (CPU, RAM, disk)
    - Network settings
    - Privileged vs unprivileged mode
    - Advanced options
  </Tab>

  <Tab title="Advanced Mode">
    For automated deployments or custom configurations:

    ```bash
    bash -c "$(wget -qLO - https://github.com/tteck/Proxmox/raw/main/ct/scriptname.sh)" -s --verbose
    ```

    Additional flags:
    - `--verbose` - Show detailed output
    - `-s` - Skip interactive prompts (use defaults)
  </Tab>
</Tabs>
